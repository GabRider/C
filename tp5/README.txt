Pour lancer les tests automatisés sur votre code:
- crééz un répertoire tp5
- dans tp5, créez un répertoire src
- dans le répertoire src, créez un répertoire à votre nom, ex: donald_knuth
- dans le répertoire à votre nom, mettez tous vos fichiers sources et votre makefile
- décompressez l'archive automated_tests.tar.gz dans le répertoire tp5
- exécutez ./run_tests
